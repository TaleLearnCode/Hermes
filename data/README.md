# Hermes Data

This directory contains the data used in the Hermes project. The data is stored in the following subdirectories:

- `engagements`: contains Chad Green's engagement data.
- `presentations`: contains Chad Green's presentation data.

## Hermes Commands

The following commands are available for working with the data:

- `hermes presentations add -i "C:\Repos\Hermes\data\presentations"`: Loads the presentation data to the Hermes project.
- `hermes engagements add -i "C:\Repos\Hermes\data\engagements"`: Loads the engagement data into the Hermes project.
