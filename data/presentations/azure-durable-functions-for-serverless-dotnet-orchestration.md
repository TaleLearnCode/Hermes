## Elevator Pitch

Discover Azure Durable Functions for serverless .NET orchestration. Learn implementation, common patterns, and best practices for efficient stateful processes.

## Short Abstract

Durable Functions is an open-source extension to Azure Functions that enables long-running orchestrations and stateful processes to execute as serverless functions. Learn about the common use patterns for durable functions, how to implement them, and the best practices for writing simple or complex stateful orchestrations.
