## Abstract

The microservice pattern serves as excellent practice in building modern applications to be more resilient and reduce dependencies across multiple system components. Often, these solutions are based on complex web frameworks and containerization. Using Azure Functions, we can develop more concise services even quicker and get many additional benefits from serverless technology without all the operational management. In this session, we will focus on how to start with an empty Visual Studio solution and build a complete serverless microservice solution using Azure Functions that solve a real-world problem. You will see firsthand how to design, develop, and deploy the Azure Function App, which is usable to perform valuable services.
