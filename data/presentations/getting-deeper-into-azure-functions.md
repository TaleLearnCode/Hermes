## Elevator Pitch

There is more to Azure Functions than just the HTTP trigger; come to see real-world examples of the different types of triggers and bindings.

## Short Abstract

You surely have seen the HTTP trigger within Azure Functions. Every presentation uses that trigger to show how awesome Azure Functions can be. But there is a bunch more you can do with Azure Functions using the other triggers and bindings, which makes Azure Functions a fantastic tool in your toolchest for solving all sorts of needs. So come and hear about these triggers and bindings and see real-world examples of how they have been employed in different solutions to solve real customer needs.
