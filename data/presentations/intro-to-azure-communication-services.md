## Elevator Pitch
Learn what the new Azure Communication Services offers and see some use cases such as sending and receiving SMS messages and two-way video conversations.

## Short Abstract
Microsoft recently announced Azure Communication Services, which “brings rich communication APIs to all your applications across any device, on any platform, using the same reliable and secure infrastructure that powers Microsoft Teams.” During this session, you will learn what this new service has to offer, and we’ll step through a couple of use cases, such as sending/receiving SMS messages and two-way video conversations.
